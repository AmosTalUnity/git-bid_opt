TJ Bid Optimization Tool

Download Processed CSV

To use the TJ Bid Optimization Tool, follow these steps:

  1. Download a Bid Manager CSV (without ROAS) and enter it in the first input field.
  2. Upload a Bid Changes CSV in the second input field. The CSV should have the following format:
    • Column A: Ad Set ID
    • Column B: Publisher App ID
    • Column C: Percentage

    Here's an example of a valid Bid Changes CSV:

    Ad Set ID Publisher App ID Percentage
    1 DEFAULT_PER_APP_BID_GROUP_ID 110
    2 42344-34343-22222 120

    In this example the bids of all the deafult apps of adset 1 will be increased by 10%, and in adset 2, the bids of app 42344-34343-22222 will be increased by 20%

  3. Click the "Process Result" button to generate the processed CSV.
  4. Upload the processed file the the CSV Bid Manager
